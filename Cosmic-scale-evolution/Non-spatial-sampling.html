Probability anomalies on evolutionary side-branches

A probability anomaly is when we observe an extremely unlikely occurrence. Anthropic sampling – selecting an ELP (Earth-Like Planed) by being born on it – distorts probabilities, so that we observe a chain of highly unprobable mutational events, leading to intelligent life.

But anthropic sampling alone does not explain similar chain of unprobable transitions on side-branches.

Eg why do we observe a diverse multitude of insects, birds, mushrooms, etc etc on earth, when they are not ‘needed’ for intelligent life?

This is a puzzle that made into my list of Millenial Problems.

The Transitional Probability is ill-defined on an infinite set of ELPs

  • Some random phenomena cannot be treated in the framework of probability theory. An example is quantum theory, where ‘events’ don’t have the distributive property.

  • A crucial tenet of probability theory is that it is impossible to define uniform probability distribution on a countably infinite set.

  • So any phenomenon that would allow uniform sampling from an infinite set cannot be assigned a probability.

  • Now consider a system with two states: A and B, and a single, random transition A→BA \to B. Initially, the system is in state A, and after time T, it is found in state B with propability TRP=P(A→B)TRP = P(A \to B).

    If you take N such systems, then the number of B-states that you will observe after time T is approximately Bcount=N*TRPB_{count} = N*TRP.

  • This allows the estimation of TRPTRP, as TRP=Bcount/NTRP = B_{count} / N.

  • Now take N→∞N \to \infty. Then Bcount→∞B_{count} \to \infty as well, so the estimation TRP=Bcount/NTRP = B_{count} / N does not work any more.

  • Why can’t we say that we select a random NN element from the infinite set, and count the BBs there, saying that this is an ‘estimation’?

    After all, opinion polls do the same all the time.

    The problem is that however large NN is, that sample will never be representative, unless we assume that they are picked randomly from the infinite set.

    And that is not possible, as it would assume a uniform distribution over an infinite set.

  • Now assume that we know the TRP. Then we can compute the probability of finding a B in any set with cardinality N: P(B)=TRP*NP(B) = TRP * N.

    However, if N is ℵ0\aleph_0, this computation breaks down.

How non-spatial sampling can yield any frequency of event B

Let me demonstrate the breakdown of probability theory first on a trivial problem.

  • Assume that we pick a random non-negative integer, using a pseudo-random number generator.

    What is the probability that we pick an even number?

  • It seems the answer is obvious: P=0.5P = 0.5

    However, it is very difficult to justify this result.

  • So if we were picking an integer from a bounded range, eg between 0 and M, where M is very large, then obviously the amount of even vs odd numbers are the same in this set, so P=0.5P = 0.5 is true.

    However, if the range is not bounded, if we allow picking any integer, then comparing the even vs odd numbers is nontrivial.

    Eg it is possible to rearrange the integers so that only every 100th is even, and two even numbers are separated by 99 odd numbers.

    Is it still P=0.5P = 0.5 that we pick an even number?

  • If we allow any linear (total) ordering of non-negative integers then it is easy to see that we can pick an ordering where any two consecutive even numbers are separated by infinitely many odd numbers: arrange the integers in a 2-dim array like below, and read out row-by-row (not diagonally!), like 2,1,3,7,...4,5,9,...2, 1, 3, 7, ... 4, 5, 9, ...

    So eg 2 and 4 are separated by infinitely many odd numbers, 4 and 6 as well, and so on:

2 1 3 7 … 4 5 9 15 … 6 11 17 25 … 8 19 27 37 … ⋮ ⋮ ⋮ ⋮ ⋰

  • Obviously, we can rearrange the positive integers so that the odd numbers become ‘infinitely rare’:

1 2 4 8 … 3 6 10 16 … 5 12 18 26 … 7 20 28 38 … ⋮ ⋮ ⋮ ⋮ ⋰

  • Conclusion: unless the even and odd numbers are arranged spatially in their usual alternating pattern, and unless we pick a random number by picking uniformly a random location, relative to this arrangement (aka ‘spatial’ sampling), there is no reason to assume that P(even)=P(odd)P(even) = P(odd)

How is this analogous of the transitional probabilities of evolutionary changes?

Back to our problem of selecting an ELP ‘randomly’ and checking if the A→BA \to B transition happened or not:

  • assume that we label each ELP with a unique positive integer; and assume that the A→BA \to B transition occurs only on planets with this integer being even.

  • Then the probability to observe the state BB is the same as the probability of picking a random positive integer that is even.

  • As this latter probability is ill-defined, so is the probability of observing the A→BA \to B transition.

  • More precisely, P(A→B)=TRPP(A \to B) = TRP only if we sample the ELPs uniformly, that is, we use spatial sampling.

    But selecting an ELP with intelligent life by being born on that planet is not spatial sampling, then we are not sampling the TRP.

  • This is the reason we observe probability anomalies, instead of observing evolutionary transitions of the highest TRP.
